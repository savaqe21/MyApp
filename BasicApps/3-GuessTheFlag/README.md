# GuessTheFlag

### Hej ![](https://user-images.githubusercontent.com/18350557/176309783-0785949b-9127-417c-8b55-ab5a4333674e.gif)

Ten projekt w SwiftUI to gra polegająca na zgadywaniu, która pomoże użytkownikom poznać niektóre z wielu flag świata. Na górze ekranu wyświetlana jest nazwa kraju, którego flagę należy znaleźć wśród trzech flag wyświetlanych poniżej. To kolejny prosty projekt, ale uczy wielu podstawowych technik SwiftUI, które zostaną wykorzystane w bardziej złożonych projektach.
